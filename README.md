# Predicting-User-Sentiment-for-Fine-Foods-products-using-Text-Summary-of-Amazon-Reviews
Accumulated data for analysis using SQL based queries from SQLite dataset using the sqlite3 package in Python.  Partitioned the review into positive and negative sentiments and cleaned the text data by stemming, tokenizing, pruning using the nltk library.
